Okta OAuth OIDC setup

  1. Visit your Okta admin panel in a browser
  2. Click Applications
  3. Click Add Application
  4. Click Web
  5. Click Next
  6. Customize Name
  7. Remove all Base URIs
  8. Change the first/only item in Login redirect URIs to “https://example.com/login” (substituting your intranet DNS domain name)
  9. Remove all Logout redirect URIs
  10. Click Done
  11. Paste the Client ID, Client secret, and Okta hostname in response to substrate-create-admin-account’s prompts
